## 🆘 Troubleshooting

### "Publish button not visible"
→ Check the top right corner of v0
→ Or look in Settings → Deploy

### "Build failed"
→ Don't worry, very rare
→ Vercel will show error details
→ Check if all files display in file tree

### "App won't load after deploy"
→ Wait 2-3 minutes for CDN to propagate
→ Clear browser cache (Ctrl+Shift+Del)
→ Try incognito/private window
